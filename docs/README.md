# Documentazione

Questa cartella contiene tutta la documentazione del progetto organizzata per categoria.

## Struttura

### `development/`
Documentazione per sviluppatori:
- **DEVELOPMENT.md** - Guida generale allo sviluppo
- **TEST_APP.md** - Come testare l'applicazione
- **TROUBLESHOOTING.md** - Risoluzione problemi comuni

### `electron/`
Documentazione specifica per Electron:
- **README.md** - Panoramica integrazione Electron
- **SETUP.md** - Guida completa setup e sviluppo Electron

### `archive/`
Documentazione storica/obsoleta:
- Analisi iniziali del progetto
- Report di sicurezza
- Analisi di design e UX
- Note di ricerca e sviluppo
